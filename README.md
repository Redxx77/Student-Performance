# 🎓 Student Performance Prediction System

##Overview

This project predicts whether a student will **Pass or Fail** based on academic and behavioral features using Machine Learning.

The system helps identify students at risk of failing so that early intervention can be taken.

---

##Features

* Predicts student performance (Pass / Fail)
* Machine Learning model (Logistic Regression)
* Interactive web app using Streamlit
* Displays confidence score
* Data visualization (graphs + confusion matrix)

---

##Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib, Seaborn
* Streamlit

---

##Dataset

Custom dataset with the following features:

* Study Hours
* Attendance
* Previous Marks
* Assignments
* Internal Marks

Target:

* Final Result (Pass / Fail)

---

##How It Works

1. Data preprocessing
2. Model training
3. Model evaluation
4. Prediction using user input
5. Display results in web interface

---

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## Output

* Prediction: PASS / FAIL
* Confidence Score
* Visualizations:

  * Study Hours vs Marks
  * Confusion Matrix

---

## Use Case

* Helps teachers identify weak students
* Enables data-driven academic decisions

---

##Limitations

* Depends on dataset quality
